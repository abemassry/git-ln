# git-ln
Open the Line number on GitHub

## INSTALL
1. make a ~/bin directory if it doesn't already exist

  `mkdir ~/bin`
  
2. add ~/bin to your PATH in your .bashrc / .zshrc / ... if it's not
   already there
   
  `export PATH=~/bin:$PATH`
   
3. copy git-ln to your ~/bin directory

  `cp git-ln ~/bin`

`git ln README.md 10`
